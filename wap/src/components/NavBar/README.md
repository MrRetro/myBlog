# 底部菜单

### Props

| 字段名            | 描述           | 类型       | 默认值 |
| ----------------- | -------------- | ---------- | ------ |
| navData | 底部菜单数据   | `Array`  | null  |
| title | 菜单操作名称   | `String`  | 菜单  |


### Events
| 字段名       | 描述     |
| ------------ | -------- |
|onChangeNav|更多菜单点击回调事件|
|onSelectById|菜单项点击回调事件|
