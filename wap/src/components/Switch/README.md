# 按钮切换

### Props

| 字段名            | 描述           | 类型       | 默认值 |
| ----------------- | -------------- | ---------- | ------ |
| v-model | 状态 | `Boolean`  | false  |
| disabled | 禁用状态 | `Boolean`  | false  |
| isWord | 文字显示 | `Boolean`  | true  |


```js
<yi-switch
  v-model="value"
  :disabled="falseß"
  :is-word="true"
/>
```
